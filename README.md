# Jada Agent

Nextcloud app for OpenClaw AI agent — autonomous tasks, skills, and MCP tools.

## Features

- **Dashboard** — Real-time agent health, activity feed, and quick actions
- **Chat** — Converse with your AI agent directly from Nextcloud
- **Skills & Tools** — Manage installed skills and MCP server connections
- **Schedules** — Create recurring or one-time automated jobs
- **Settings** — Configure OpenClaw connection, model selection, and identity

## Requirements

- Nextcloud 28+
- PHP 8.1+
- OpenClaw instance (self-hosted Docker container)
- API key for your LLM provider (OpenRouter, Anthropic, OpenAI, etc.)
